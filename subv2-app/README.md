# React-Webpack-Template

## Introduction

React17 template with Webpack5

## Getting Started

```javascript
# enter the project directory
cd react-webpack-template

# install dependency
yarn or npm install

# develop
yarn start or npm run start

```

## Features

- React 17
- typescript
- Webpack 5
- @reduxjs/toolkit
- Css-Modules
- less
- Eslint
- commitlint
- stylelint
